Year 3 Event 18 No-Cost Extension

Story Background

More time may be needed, but time must be justified

The project is approaching its scheduled end date. Much of the Community Access to Preventive Healthcare Initiative has been completed, but not everything is finished.

Dr. Brown wants additional time to complete final community outreach activities, finish data collection follow-up, receive final CHAN documentation, and prepare the project’s final reports.

She is not asking for additional funding. Instead, she wants to extend the project period so the remaining approved work can be completed using funds already awarded.

The question is not simply whether money remains. Marcus must help determine whether additional time is needed to complete approved project work and whether the request can be routed through the required sponsor process before the project period ends.

Trigger Issue

Dr. Brown asks about extending the project period

From: Dr. Aneka Brown

To: Marcus Rivera

Subject: No-Cost Extension Request

Hi Marcus,

We are approaching the end of the current project period, and I would like to request a no-cost extension.

We do not need additional funds, but we do need additional time to complete final outreach activities, reconcile CHAN’s remaining work, and finish the final data collection follow-up.

Can you help determine what we need to submit and whether our remaining funds can support the extension period?

Thank you,
Dr. Aneka Brown

Training Concept

A no-cost extension adds time, not money

A no-cost extension extends the project period without adding new sponsor funds. It may be appropriate when more time is needed to complete approved project activities. It should not be used simply because money remains.

Project Period Extension

Additional time added to the approved project period so remaining work can be completed.

No Additional Funds

The sponsor does not provide new money. The project uses remaining awarded funds.

Remaining Work

The request should explain what approved project activities still need to be completed.

Remaining Funds Alone Are Not Enough

Having money left is not a sufficient reason to extend an award.

Justification

The request should explain why the extension is needed and how the time will be used.

Sponsor Requirements

The Notice of Award or sponsor guidance determines the process, deadline, required content, and approval path.

Marcus’s Analysis

Questions before routing the request

What approved work remains?

Marcus confirms that the remaining activities are part of the approved project and not new scope.

Why was more time needed?

The request should explain the reason for the delay or unfinished work in a clear, factual way.

Can the funds support the extension?

Marcus reviews remaining funds, pending costs, payroll needs, subrecipient costs, and closeout expenses.

Does the extension affect reporting?

New reporting dates, final deliverables, and closeout timelines may change if the extension is approved.

Does CHAN need more time too?

If CHAN has remaining work, the subaward timeline may also need review.

What does the sponsor require?

Marcus checks the deadline, form, justification, signatures, and institutional routing path required before submission.

Decision Point

Should Marcus support routing the no-cost extension request?

Dr. Brown asks for additional time to complete final project activities. The project has remaining funds. What should Marcus recommend?

Resolution

Marcus prepares the request for review with a clear justification

Marcus determines that the request has a reasonable administrative basis because approved work remains, the remaining funds can support the proposed extension activities, and the request can be submitted before the sponsor deadline. The request still needs to follow the required institutional and sponsor process.

Records Reviewed Notice of Award, project timeline, remaining funds report, pending obligations, CHAN status, final deliverables, and sponsor extension requirements.

A strong no-cost extension request explains what remains, why more time is needed, how existing funds will be used, and how the project will reach completion.

Common Misconceptions

What learners should not assume

“No-cost extension means free extra time.”

The extension must follow sponsor requirements and be justified by remaining approved work.

“Remaining funds are enough reason to extend.”

Remaining funds alone do not justify an extension. The project must need time to complete approved activities.

“We can request it after the award ends.”

Sponsors usually require extension requests before the project end date. Marcus must check the deadline and process early.

2 CFR 200.308 — Revision of Budget and Program Plans

This section matters when a project needs changes to the approved budget, program plan, or project period. Depending on the award terms, a no-cost extension may require sponsor notification or prior approval.

Connection to Event 18: Marcus reviews whether extending the project period requires sponsor approval before the team continues work beyond the current end date.
